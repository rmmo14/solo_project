Welcome to Math+C: About This Site

Let's Collaborate!

Work with a community of users to solve various levels of math problems. If there is a solution posted, and the question is solved, still contribute to show multiple ways to the same solution. If you do not agree with a posted solutin, refute it and if a minimum of 3 users refute it then that solution will be deleted.

Future Updates

In the coming weeks, the site will show a chart for the history of each solutions agree and refute record. There will also be improvements on agreeing where it does not allow the posting user to agree to their own solution. Eventually we will be integrating math text programming so users can copy and paste their problems when neeeded.

Example of Subjects

For the moment, we will be hosting questions for topics in College Algebra, Precaclculus, Calculus 1, and Calculus 2. With the updates to come, there will be room for a wider range of math subjects.
